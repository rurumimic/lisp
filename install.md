# Install

## OS

### Ubuntu

```bash
sudo apt insatll sbcl
sudo apt install clisp
```

### macOS

```bash
brew install clisp
```

---

## Run

### sbcl

```bash
sbcl
```

```lisp
* (princ "Hello World")
Hello World
"Hello World"

* (quit)
```

### clisp

```bash
clisp
```

```lisp
[1]> (princ "Hello World")
Hello World
"Hello World"

[2]> (quit)
```
